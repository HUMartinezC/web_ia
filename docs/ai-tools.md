# Herramientas de IA

## 2026-04-06

- Copilot Chat:
	- Se utilizó para planificar por fases, generar código y refinar funcionalidades del proyecto.
	- Fue clave para iterar rápidamente en Angular, integración con Hugging Face y documentación.
- Hugging Face Router:
	- Se usó para generación de cuestionarios, clasificación por categorías y generación de imágenes por quiz.
- Vercel Functions:
	- Se usó como capa serverless para proteger el token de Hugging Face mediante variable de entorno.

## Valoración general

- Productividad: alta.
- Calidad de iteración: alta, especialmente en ciclos cortos de prueba/corrección.
- Riesgos detectados: necesidad de validar formatos de respuesta IA y manejar fallbacks.
