# Generar Web 

Diseñar, desarrollar y presentar una página web usando mayoritariamente IA (vibe coding).

## Objetivos

- Web educacional donde un usuario escribe un tema y un modelo de Hugging Face devuelve un cuestionario según un nivel de dificultad (fácil, normal, difícil) mediante un selector.

- La web será simple, con un sidebar con opciones para ir a la creación de cuestionarios (promt), ver los guardados, repetirlos, etc.

- Podría haber también un apartado que guarde todas las respuestas y muestre métricas simples: tests hechos, porcentace global de acierto de preguntas, además, un segundo modelo de Hugging Face podría clasificar los tests por categorías.

- Es estilo visual será algo simple pero divertido. (desarróllalo)

- La web se desplegará en Vercel.

- Todo el desarrollo, en principio, será realizado por una IA.

- Se creará documentación en `docs/`, en formato Markdown.

## Tecnologías

- Se utilizará Angular 21 con la forma recomendada de desarrollo.
- Gestión de estado mínima con herramientas nativas.
- La estructura de carpetas debe ser simple y funcional.

## Requisitos originales de la práctica a desarrollar

Requisitos mínimos:
- Cada integrante debe participar activamente y documentar sus aportes.
- Todos los prompts deben estar documentados y justificar cómo contribuyen al proyecto.
- La página debe poder ejecutarse en un navegador y mostrar correctamente el contenido y la funcionalidad.
Imágenes y vídeos generados por IA, integrados coherentemente en la web.
- Al menos debe tener un modelo de IA de hugging face que se use mediante inferencia.
- La web debe ser creativa, con identidad propia y diferenciable.
- Explicación de herramientas usadas, prompts clave y reflexiones sobre lo aprendido.

Debe entregarse los siguientes documentos:
- Enlace de Aplicación Web
- Documentación realizada
- Presentación del proyecto a la clase, debéis indicar como la habéis realizado y  mostrar funcionalidades.

Recomendaciones:
- Exploren herramientas de para generación de código, diseño y multimedia.
- Experimenten con distintos prompts y versiones hasta obtener el resultado deseado.
- La integración final debe ser coherente, optimizada y funcional
